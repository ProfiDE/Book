# ProfiDE Book

> [!IMPORTANT]
> This is a magical and useful book, so don't expect a strict structure. You are welcome to read it and even help improve it by opening an issue and submitting a pull request (please reference your issue).

An ever-evolving book that covers a wide range of topics, from IT to simple stories. Focused on useful subjects including programming, networking, security, and more.

## Contributing

Thank you for considering expanding the content of this book!

To add pages, edit content, or suggest ideas, **please start by opening an issue** and describe the changes you wish to make.

Submitting a pull request that references an issue will help us review and apply your changes more easily and you can add your name to contributors list. Please avoid altering the base structure or making changes beyond what is noted in the issue.

If you plan to modify or add multiple pages, open individual issues for each and reference them all in the same pull request.
